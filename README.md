# Superstore Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard created using the Superstore 
dataset to analyze sales performance, profit trends, and regional sales distribution.

## Objectives
- Analyze overall sales performance
- Identify top performing regions
- Track monthly sales and profit trends
- Understand sales by category and segment

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Dashboard Features
- Sales by Region
- Sales by Segment
- Sales by Category and Sub-Category
- Monthly Sales Trend
- Monthly Profit Trend
- Sales by Ship Mode

## Dataset
The dataset contains information related to orders, sales, profit, customer segments, and shipping modes.

## Project Files
- Superstore_Sales_Dashboard.pbix (Power BI Dashboard)
- Superstore_Dataset.csv (Dataset)
